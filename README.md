# Portfólio de Machine Learning
Esse projeto reúne modelos de ML desenvolvidos por mim, organizados em Classificação, Regressão e Séries Temporais. Execute localmente com `streamlit run app.py` ou veja online via Render!

## Rodando Localmente
1. Instale dependências: `pip install -r requirements.txt`
2. Salve os modelos treinados em `/modelos`
3. Inicie: `streamlit run app.py`
